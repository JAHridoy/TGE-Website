# Titas Global Education — The Academic Curator

![Platform Preview](/assets/homepage_hero.png)

## Overview
**Titas Global Education** is a premium, high-density web application designed for the modern international student. Built with the "Academic Curator" aesthetic, the platform bridges the gap between ambitious scholars and world-class institutions through a sophisticated, data-driven consultancy ecosystem.

### Key Pillars
- **Strategic Profiling**: Deep-dive academic matching for elite destinations.
- **Narrative Excellence**: Personalized SOP and portfolio coaching that goes beyond grades.
- **Ethical Architecture**: Radical transparency in visa processing and university pathways.

---

## 🎨 Design Philosophy
The application adheres to a "Sophisticated Authority" visual language:
- **Glassmorphism**: Subtle translucent layers reflecting modern transparency.
- **Material Symbols**: Precise, institutional iconography for clarity.
- **Inter & Serif Typography**: A balance of modern readability and timeless academic heritage.
- **Premium Photography**: High-resolution, brand-aligned assets representing global landmarks and diverse student life.

---

## 🚀 Tech Stack
- **Core**: React 18 + TypeScript + Vite
- **Styling**: Tailwind CSS (Custom Design System)
- **Routing**: React Router DOM v6
- **Iconography**: Google Material Symbols (Rounded)
- **Asset Management**: Local Optimized Assets & Unsplash API

---

## 📂 Project Structure
- `/src/pages`: Destination-specific deep dives (UK, Canada, Australia, Cyprus, etc.)
- `/src/components`: Reusable UI components (Navbar, Footer, Glass-sections)
- `/public/assets`: Premium local photographic assets.

---

## 🛠️ Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
